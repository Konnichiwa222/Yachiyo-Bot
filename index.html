What it does
Everything your server needs
/antilink

Anti-Link

Automatically detect and remove unwanted links posted by members, keeping your server clean and safe.

/autoresponder

Auto Responder

Set custom keyword triggers and let the bot reply automatically — perfect for FAQs and greetings.

/autorole

Auto Role

Assign roles to new members automatically when they join, or based on other conditions you configure.

/giveaway

Giveaways

Host giveaways with ease. Set duration, prize, and number of winners — the bot handles the rest.

/ticket

Ticket System

Let members open support tickets through interactive panels. Organized, private, and easy to manage.

/streak

Streak Tracking

Reward active members by tracking daily message streaks and building a culture of engagement.

/lock

Channel Lock

Lock and unlock channels instantly during emergencies or maintenance with a single command.

/afk

AFK Status

Members can set an AFK message so others know they're away. Automatically cleared when they return.

/help

Fun Commands

Keep things lively with avatar lookup, gay rate, Minecraft server status, and more playful commands.

Getting Started

How to Use

Invite Bot

2

Configure

Use /setup or /config to customize bot settings for your server needs.

3

Set Permissions

Enjoy!

Start using commands like /help to see all available features and have fun!
