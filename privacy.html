Privacy Policy

Your Privacy

Last Updated: March 15, 2026 Effective: March 15, 2026
Section 01
Data We Collect

Yochiyo Hub collects only the data necessary for the bot to function:

  • Server ID & Channel IDs — For server configuration and channel-specific features.
  • User IDs — To track user-specific data like streaks, points, and permissions.
  • Message content — Only when required for automod, logging, or specific commands you explicitly use.
  • Interaction data — Button clicks, modal submissions, and command usage.
Section 02
How We Use Your Data

Your data is used solely to provide and improve our services:

  • To moderate your server using automod and filter features.
  • To track user engagement like streaks, points, and leaderboards.
  • To provide giveaways and interactive features.
  • To log events as configured by server owners.
Section 03
Data Storage & Security
  • All user data is stored in encrypted databases.
  • We implement industry-standard security measures to protect your data.
  • Data is stored on secure servers with limited access.
  • We do not sell your data to third parties.
Section 04
Data Retention

We retain data only as long as necessary:

  • Server data — Retained until the bot is removed from the server.
  • User data — Retained while the user remains active on servers using the bot.
  • Logs — Configurable retention period set by server owners (default: 30 days).
Section 05
Third-Party Services
Yochiyo Hub uses the following third-party services:
  • Discord API — For bot functionality and user interactions.
  • Database Hosting — For persistent data storage.
Section 06
User Rights

You have the following rights regarding your data:

  • Access — Request a copy of your stored data.
  • Deletion — Request removal of your data (use /request-data-deletion command).
  • Opt-out — Leave any server using the bot to stop data collection for that server.
Section 07
Children's Privacy
Yochiyo Hub is not intended for users under 13 years of age. We do not knowingly collect data from children under 13.
Section 08
Changes to This Policy

We may update this Privacy Policy periodically. We will notify users of any material changes via our support server or Discord DM.

Questions? Contact us directly via Discord.
